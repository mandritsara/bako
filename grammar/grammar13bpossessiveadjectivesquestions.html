Malagasy–Tsimihety Grammar Lesson 13B

Possessive Adjectives: Questions

May 2025 — Malagasy–Tsimihety with Bako

1. Review: Building a Possessive Adjective

The structure is the same as in Lesson 13A:

NOUN (thing possessed) + SUFFIX (possessor)

Examples:

  • Mofo (off) / môfo (tsm) + ZAHAY (we excl.)  →  MofoNAY (our bread)
  • Latabatra + ANTSIKA (we incl.)  →  LatabatraTSIKA (our table)
  • Penina + IZY (he/she)  →  PeninANAZY (his/her pen)
  • Satroka (off) / satroko (tsm) + ZAHO (I)  →  SatrokoANAHY (my hat)

2. Asking 'Whose?' — The Possessive Question

The question word is a suffix: -IZA (off) / -ZÔVY (tsm), meaning WHOSE.

NOUN + IZA? (off)   /   NOUN + ZÔVY? (tsm)

Example:

  • Môfonjôvy tôho? (tsm) / MofoN'IZA ity? (off)  →  Whose bread is this?
  • Answer: MofoNAY  (our bread)

3. Dialogue Practice

Listen and repeat the dialogues below. Then practise the same dialogues, substituting:

  • In dialogue A: lobaka (shirt), kapa (sandals), mpanjaitra (tailor)
  • In dialogue B: sakafo (meal / food), dite (tea)

Dialogue A:

— Satrok'IZA ity? (Whose hat is this?)

— SatroKO io. (That is my hat.)

— Ary kiraron'IZA ireto? (And whose shoes are these?)

— KiraroN'I SOA ireo. (Those are Soa's shoes.)

— Ary mpianatr'IZA izy? (And whose student is he?)

— MpianaTRY NY mamako. (My mother's student.)

Dialogue B:

— TranonjÔVY tôho? (Whose house is this?)

— TranoNAY. (Our house.)

— Ary reto? (And these?)

— TranoN'NY dokotera. (The doctor's house.)

— Ary biraonjÔVY tôho? (Whose office is this?)

— BiraoNJARE(O). (Their office.)

4. Exercise — Answer the Questions

For each row, form the possessive question and answer using the noun and possessor given.

Reference words: ity / tôho (this), izany (that), ireto (these), izy (he/she)
