2. Useful Phrases

Learn the following phrases to help set your newly acquired vocabulary into sentences:

Aiza ny…? — Where is the…?

E.g. Aiza ny latabatra? (Where is the table?)

Tondroy ny… — Point to the…

E.g. Tondroy ny latabatra. (Point to the table.)

3. Verbs — Cooking Actions

Manasa = wash

Manadio = clean

Mamaofyoff / magnofytsm = peel

Mitetikaoff / manapatapakatsm = chop

Mahandro = cook

Manendy = fry

Mitonooff / mitognotsm = grill

Manao / mitsabo = make

4. Sentences

To set the verbs into sentences, start by asking:

Maninona izyoff / Manino izytsm? — What is she/he doing?

Henoy dia averimbereno avy eo adikao (Listen, repeat, and copy)

Manasa kapila zaho = I wash the dishes

Manadio lakozia (i)anao = You clean the kitchen

Magnofy karaoty izy = She peels the potatoes

Mitetika ovy zahay = We chop the vegetables

Mahandro vary antsika = We cook rice

Manendy hena areo = They fry meat

Mitono akoho zareo = They grill chicken
